# Make an algorithm that reads the price of a product and displays its new price, with 5% off.

## Example in Python

### Code

``` python
price = float( input(' type the price: ') )
print(' the price with 5% off is {:.2f}'.format(price * .95) )

```

### Output

```
 type the price: 20
 the price with 5% off is 19.00

```
